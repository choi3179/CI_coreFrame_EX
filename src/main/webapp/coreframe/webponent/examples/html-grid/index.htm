WebPonent Ajax Grid

개요

WebPonent Ajax Grid는 기존 HTML table 의 기능을 확장하기 위하여 Html 양식의 table 을 간단한 javascript 함수를 이용하여 RIA 수준의 Grid를 구현하기 위한 HTML/JavaScript 기반의 Grid 이다.

기본적인 디자인이나 데이터 양식은 HTML 의 Table 태그를 그대로 이용하여 구현하고, new webponent.HtmlGrid('테이블ID', '가로크기', '세로크기'); 와 같은 방식으로 기존의 table을 확장한다.

  • 헤더가 고정된 상태에서 가로/세로축 scrolling 기능
  • 컬럼폭 resize 기능
  • 컬럼 타입별 sort 기능
  • AJAX 방식에 의핸 외부 데이터 호출기능
  • 오른쪽 팝업메뉴기능
  • 다중 row 선택 기능
  • 엑셀출력기능
  • 웹표준 준수

사용법

  1. 표현하고자 하는 그리드 모양을 일반적인 표준 HTML table를 작성한다.
  2. 작성한 HTML table은 반듯이 고유한 id를 부여한다.
  3. HMTL document 로드 완료후 원하는 html를 Webponent Grid로 변환하기 위하여 아래와 같이 Javascript API 를 이용하여 호출한다.
  4. var grid = new webponent.HtmlGrid('tabldID', '600px', '300px',{ 확장 options});
  5. 확장옵션은 {'option1':'value1', 'option2':'value2'} 형태와 같은 JSON 형태로 기술한다.
